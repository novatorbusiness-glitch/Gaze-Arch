1. Быстрая навигация (Маст-хэв)
Cmd + P (Mac) / Ctrl + P (Win): Поиск файла по названию. Просто начни вводить style или index, чтобы мгновенно переключиться.

Cmd + Shift + F / Ctrl + Shift + F: Глобальный поиск по всему проекту. Помогает найти, в каких файлах ты использовал конкретную переменную или класс.

Cmd + Option + ← / →: Быстрый переход между открытыми вкладками.

Ctrl + G: Переход на конкретную строку кода.

2. Умное редактирование (Работа с кодом)
Cmd + D / Ctrl + D: Выделение следующего такого же слова. Идеально, если нужно переименовать класс сразу в нескольких местах блока.

Option + ↑ / ↓ / Alt + ↑ / ↓: Переместить текущую строку (или выделенный блок) вверх или вниз. Не нужно копировать и вставлять — просто двигай код.

Shift + Option + F / Shift + Alt + F: Форматирование кода. Выравнивает все отступы и скобки по стандарту (Lighthouse это любит).

Cmd + / / Ctrl + /: Закомментировать или раскомментировать строку.

3. Работа с терминалом и интерфейсом
**Ctrl + \ (тильда):** Открыть/скрыть встроенный терминал. Тот самый, где мы пишемgit push`.

Cmd + B / Ctrl + B: Скрыть/показать боковую панель. Нужно, чтобы освободить больше места для верстки.

Cmd + J / Ctrl + J: Скрыть нижнюю панель (терминал, ошибки).

4. Специфические команды для Cursor (AI)
Так как ты работаешь с ИИ, эти команды — твои основные инструменты:

Cmd + K / Ctrl + K: Генерация или правка кода в выделенном фрагменте.

Cmd + L / Ctrl + L: Открыть чат с ИИ, который видит весь твой проект (если нажать Enter с зажатым Cmd/Ctrl).

Cmd + I: Открыть Composer (для правок сразу в нескольких файлах).

5. Команды «Спасения»
Cmd + Shift + P / Ctrl + Shift + P: Командная палитра. Если забыл горячую клавишу, просто нажми это и начни писать действие (например, Reload Window или Timeline).

Developer: Reload Window: Перезагрузка VS Code. Лечит 90% глюков интерфейса или расширений.
